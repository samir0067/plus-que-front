## Description

## Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/samir0067/plus-que-front.git
   cd plus-que-front
2. Installez les dépendances :
    ```bash
      npm install
    ```
3. Configurez les variables d'environnement dans un fichier .env
   ```bash
      VITE_API_KEY=YOUR_API_KEY
   ```

4. Lancez l'application :
    ```bash
      npm run dev
    ```
